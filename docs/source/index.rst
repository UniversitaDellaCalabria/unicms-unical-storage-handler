uniCMS storage.unical.it APIs handler
=====================================

This `uniCMS <https://unicms.readthedocs.io/en/latest/index.html>`_ app
allows you to interface with the public
`storage.unical.it <https://github.com/UniversitaDellaCalabria/storage.portale>`_
APIs making users to consume information in a graphically appealing format.

The ability to customize settings file parameters and the ease of
building new templates make this handler perfect for developers wishing
to make their uniCMS instance dynamic and powerful.
